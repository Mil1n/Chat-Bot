# Промт для проекта «AI/FAQ-чат-бот в браузере»

```text
Ты — опытный frontend/backend-разработчик и наставник. Создай учебный AI/FAQ-чат-бот, который работает в браузере и может опционально подключаться к AI backend.

Сделай файлы:
- index.html — интерфейс чата;
- styles.css — современный адаптивный дизайн, светлая/тёмная тема;
- script.js — логика frontend;
- faq.json — FAQ и мнения бота;
- knowledge.json — простая RAG-база знаний;
- server.js — Node.js backend с endpoint /api/chat;
- README.md — инструкция запуска.

Требования:
1. Бот работает локально без AI API: берёт ответы из faq.json и knowledge.json.
2. Бот сохраняет историю диалога в localStorage.
3. Добавь кнопку очистки истории.
4. Добавь светлую/тёмную тему с сохранением выбора.
5. Добавь несколько характеров бота: дружелюбный, экспертный, строгий, смешной.
6. Добавь режим «своего мнения» по триггерам: «мнение», «думаешь», «как считаешь».
7. Добавь быстрые кнопки популярных вопросов.
8. Добавь оценку ответов кнопками 👍/👎.
9. Вынеси FAQ из JS в JSON и загружай через fetch().
10. Добавь простую RAG-логику: поиск подходящего документа по ключевым словам из knowledge.json.
11. Backend должен хранить AI API ключ только на сервере, читать OPENAI_API_KEY из переменных окружения и не раскрывать ключ в браузере.
12. Если AI API недоступен, frontend должен использовать локальный fallback.
13. Код должен быть понятным для новичка и без тяжёлых фреймворков.

После кода объясни:
- откуда бот берёт ответы;
- чем локальный режим отличается от AI-режима;
- почему API-ключ нельзя хранить во frontend;
- как развивать проект дальше.
```
