# Промт для проекта «Умный чат-бот в браузере»

Скопируй этот промт в ChatGPT/Codex, чтобы получить учебный проект браузерного FAQ-чат-бота.

```text
Ты — опытный frontend-разработчик и наставник для начинающих. Создай умного FAQ-чат-бота, который работает прямо в браузере без сервера и внешних API.

Цель проекта: потренировать логику, условия, работу со строками, массивами, объектами, DOM-событиями и простым интерфейсом.

Требования:
1. Создай три файла: `index.html`, `styles.css`, `script.js`.
2. Интерфейс должен выглядеть как современный чат: область сообщений, поле ввода, кнопка отправки и 2-3 быстрые кнопки с готовыми вопросами.
3. Бот приветствует пользователя при открытии страницы.
4. Бот отвечает минимум на 8 популярных FAQ-тем из заранее заданной базы.
5. Вопросы пользователя нужно обрабатывать без учёта регистра и буквы «ё».
6. Бот должен быть «умным» для учебного проекта: ищи лучший ответ по ключевым словам и выбирай тему с максимальным числом совпадений.
7. Добавь режим «своего мнения»: если пользователь спрашивает «что ты думаешь», «твоё мнение», «как считаешь», бот отвечает с аккуратной позицией, а не только фактом.
8. Бот должен понимать похожие формулировки через ключевые слова, например:
   - «привет», «здравствуй», «hello»;
   - «пока», «выход», «стоп»;
   - «цена», «стоимость», «сколько стоит»;
   - «браузер», «сайт», «web».
9. Если бот не знает ответ, он вежливо сообщает об этом и предлагает список тем, по которым может помочь.
10. Код должен быть понятным для новичка:
   - используй функции;
   - добавь комментарии к важным частям;
   - не используй фреймворки и сложные библиотеки.
11. После кода кратко объясни, как он работает и откуда берутся ответы.
12. В конце предложи 3 идеи, как улучшить проект: настоящий AI API через backend, память через localStorage, загрузка FAQ из JSON.

Формат ответа:
- сначала полный код `index.html`;
- затем полный код `styles.css`;
- затем полный код `script.js`;
- затем инструкция запуска в браузере;
- затем объяснение логики;
- затем идеи для развития.
```

## Что должно получиться

После выполнения промта ожидается небольшой сайт с чат-ботом. Его можно открыть напрямую через `index.html` или запустить локальный сервер:

```bash
python3 -m http.server 8000
```

Затем открыть:

```text
http://localhost:8000
```
