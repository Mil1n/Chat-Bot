# 🧭 Roadmap Chat-Bot

## ✅ Сделано

1. **Память диалога через `localStorage`** — история сохраняется после перезагрузки.
2. **Больше быстрых кнопок** — добавлены вопросы про старт, мнение, RAG и AI API.
3. **Переключатель темы** — светлая и тёмная тема сохраняются локально.
4. **FAQ из отдельного JSON-файла** — ответы вынесены в `faq.json`.
5. **Оценка ответов** — под ответами есть кнопки 👍/👎.
6. **Настройки характера бота** — дружелюбный, экспертный, строгий и смешной режимы.
7. **Настоящий AI API через backend** — добавлен `server.js` с endpoint `/api/chat`.
8. **Системный промт с личностью бота** — backend передаёт выбранный характер в AI-модель.
9. **RAG по своим документам** — добавлен простой поиск по `knowledge.json`.

## 🔜 Что можно улучшить дальше

- сделать красивую панель статистики оценок;
- добавить редактор FAQ прямо в интерфейсе;
- подключить полноценную векторную базу для RAG;
- добавить авторизацию и разные истории для разных пользователей;
- покрыть логику unit-тестами.
