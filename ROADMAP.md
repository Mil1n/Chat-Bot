# 🧭 Roadmap Chat-Bot

## ✅ Сделано

1. Память диалога через `localStorage`.
2. Расширенные быстрые кнопки.
3. Светлая и тёмная тема.
4. FAQ из `faq.json`.
5. Оценка ответов 👍/👎 и статистика.
6. Настройки характера бота.
7. Node.js backend с `/api/chat`.
8. Системный промт с личностью бота.
9. RAG-поиск по `knowledge.json`.
10. Админка для добавления FAQ из браузера.
11. Профиль пользователя с именем.
12. Редактор своей личности бота.
13. Уровень уверенности ответа.
14. Режим спора/дискуссии.
15. Загрузка `.txt/.md` файлов в базу знаний.
16. Источники в ответах.
17. Упрощённый keyword/vector-like поиск по словам.
18. Анимация печати.
19. Голосовой ввод.
20. Экспорт диалога в Markdown.

## 🔜 Что можно улучшить потом

- заменить keyword-поиск настоящими embeddings;
- добавить базу данных и аккаунты пользователей;
- сделать полноценную админ-панель с сохранением на сервер;
- добавить unit-тесты и CI.
