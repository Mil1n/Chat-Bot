# 💡 Следующие идеи для доработки

## ✅ Реализовано сейчас

- Админка для редактирования FAQ в браузере.
- Статистика качества ответов по 👍/👎.
- Память пользователя: имя, характер, история, настройки.
- Редактор личности бота.
- Уровень уверенности ответа.
- Режим спора/дискуссии.
- Загрузка `.txt` и `.md` файлов в локальную базу знаний.
- Источники рядом с ответами.
- Упрощённый keyword/vector-like поиск.
- Анимация печати.
- Голосовой ввод через Web Speech API.
- Экспорт диалога в Markdown.

## 🚀 Новые идеи на будущее

1. **Настоящие embeddings** — заменить поиск по ключевым словам на семантический поиск.
2. **Серверное хранение** — сохранять FAQ, оценки и документы в базе данных.
3. **Аккаунты пользователей** — разные истории и настройки для разных людей.
4. **CI и тесты** — покрыть scoring, localStorage и backend unit-тестами.
5. **Деплой** — опубликовать проект на Render/Vercel/Fly.io.
