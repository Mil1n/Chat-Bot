# 🤖 Chat-Bot — AI/FAQ-бот в браузере

> Учебный проект: чат-бот на **HTML + CSS + JavaScript** с памятью, темами, характерами, FAQ-админкой, загрузкой знаний, оценкой ответов и опциональным AI backend.

## ✨ Что умеет бот

- 💬 отвечает в формате живого чата с анимацией печати;
- 🧠 выбирает лучший FAQ-ответ по ключевым словам и показывает уверенность;
- 😎 имеет режим «своего мнения», дискуссионный режим и редактор личности;
- 👤 запоминает имя пользователя и любимый характер;
- 💾 сохраняет историю диалога, настройки и оценки в `localStorage`;
- 🌗 переключает светлую/тёмную тему;
- 📚 берёт FAQ из `faq.json`, знания из `knowledge.json` и загруженных `.txt/.md` файлов;
- 🛠️ позволяет добавить FAQ-тему прямо из браузера;
- 👍 собирает оценки ответов и показывает простую статистику;
- 🎙️ поддерживает голосовой ввод через Web Speech API, если браузер это умеет;
- ⬇️ экспортирует диалог в Markdown;
- 🤖 умеет обращаться к `/api/chat`, если запустить `server.js` и задать AI API ключ.

## 🧩 Откуда бот берёт ответы?

1. **Локальный режим**: `script.js` загружает `faq.json` и `knowledge.json`, добавляет пользовательские FAQ/файлы из `localStorage`, считает совпадения ключевых слов и выбирает лучший ответ.
2. **RAG-поиск**: бот ищет подходящий документ в базе знаний и показывает источник рядом с ответом.
3. **AI-режим**: если доступен `server.js` с `OPENAI_API_KEY`, браузер отправляет вопрос на `/api/chat`, а сервер безопасно проксирует запрос к модели.

⚠️ API-ключ нельзя хранить в frontend: его увидит любой пользователь браузера. Поэтому ключ используется только на сервере.

## 🚀 Запуск без AI

```bash
python3 -m http.server 8000
```

Открой:

```text
http://localhost:8000
```

## 🤖 Запуск с AI backend

```bash
OPENAI_API_KEY=your_api_key_here npm start
```

Можно выбрать модель:

```bash
OPENAI_API_KEY=your_api_key_here OPENAI_MODEL=gpt-4.1-mini npm start
```

## 🧪 Что проверить

- добавь своё имя и переключи характер;
- включи режим дискуссии и спроси: `Поспорь со мной про Telegram-бота`;
- загрузи `.txt` файл в базу знаний и спроси по его теме;
- добавь FAQ через админку и задай вопрос по новым ключевым словам;
- поставь 👍/👎 и посмотри статистику;
- экспортируй диалог в Markdown.
