# 🤖 Chat-Bot — умный бот в браузере

> Учебный проект: красивый FAQ/ИИ-бот, который работает прямо в браузере на **HTML + CSS + JavaScript** без сервера.

## ✨ Что умеет бот

- 💬 отвечает в формате живого чата;
- 🧠 выбирает лучший ответ по ключевым словам и счёту совпадений;
- 😎 имеет режим «своего мнения» — если спросить «что ты думаешь», «твоё мнение», «как считаешь», бот отвечает не сухо, а с позицией;
- 👋 понимает приветствия и прощания;
- 🔎 подсказывает темы, если не знает точный ответ;
- ⚡ работает полностью в браузере без установки зависимостей;
- 📱 адаптирован под телефон и компьютер.

## 🧩 Откуда бот берёт ответы?

Сейчас бот **не подключён к настоящей нейросети** и не ходит в интернет. Все ответы лежат прямо в файле `script.js`:

- массив `faq` хранит темы, ключевые слова и готовые ответы;
- массив `opinions` хранит темы, по которым бот может высказать «своё мнение»;
- функция `findBestAnswer()` нормализует вопрос пользователя, считает совпадения по ключевым словам и выбирает самый подходящий ответ.

Это удобно для обучения: можно открыть `script.js`, добавить новую тему, новые ключевые слова — и бот сразу станет отвечать шире.

## 🧠 Как сделать настоящего ИИ-бота?

Чтобы бот реально генерировал новые ответы, спорил, рассуждал и имел более гибкую позицию, ему понадобится подключение к LLM/API:

1. 🖥️ **Frontend** — текущая страница чата в браузере.
2. 🔐 **Backend** — небольшой сервер, который хранит API-ключ и не показывает его пользователям.
3. 🤖 **AI API** — модель получает сообщение пользователя, системный промт с характером бота и возвращает ответ.
4. 🧭 **Персона бота** — отдельный промт, где описано: стиль общения, границы, мнение, тон и темы.

⚠️ Важно: API-ключ нельзя хранить в `script.js`, потому что любой пользователь сможет увидеть его в браузере.

## 🚀 Как запустить

Открой файл `index.html` в браузере или запусти локальный сервер:

```bash
python3 -m http.server 8000
```

Затем перейди по адресу:

```text
http://localhost:8000
```
