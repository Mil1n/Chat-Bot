# 🤖 Chat-Bot — AI/FAQ-бот в браузере

> Учебный проект: красивый чат-бот на **HTML + CSS + JavaScript** с памятью, темами, характерами, FAQ из JSON, оценкой ответов и опциональным AI backend.

## ✨ Что умеет бот

- 💬 отвечает в формате живого чата;
- 🧠 выбирает лучший FAQ-ответ по ключевым словам и счёту совпадений;
- 😎 имеет режим «своего мнения» и несколько характеров;
- 💾 сохраняет историю диалога в `localStorage`;
- 🌗 переключает светлую/тёмную тему;
- 📚 берёт FAQ и мнения из `faq.json`, а мини-базу знаний — из `knowledge.json`;
- 👍 собирает оценки ответов;
- 🤖 умеет обращаться к `/api/chat`, если запустить `server.js` и задать AI API ключ;
- 📱 адаптирован под телефон и компьютер.

## 🧩 Откуда бот берёт ответы?

Есть два режима:

1. **Локальный режим без AI** — работает сразу в браузере.
   - `faq.json` хранит FAQ-темы, ключевые слова и готовые ответы;
   - `knowledge.json` хранит маленькую RAG-базу знаний;
   - `script.js` загружает JSON, нормализует вопрос, считает совпадения и выбирает лучший ответ.
2. **AI-режим через backend** — включается, если запустить `server.js` и передать `OPENAI_API_KEY`.
   - браузер отправляет вопрос на `/api/chat`;
   - сервер добавляет системный промт с характером бота;
   - AI-модель генерирует ответ и возвращает его в интерфейс.

⚠️ API-ключ нельзя хранить в `script.js`: его увидит любой пользователь браузера. Поэтому ключ используется только на сервере.

## 🚀 Как запустить локально без AI

```bash
python3 -m http.server 8000
```

Затем открой:

```text
http://localhost:8000
```

## 🤖 Как запустить с AI backend

```bash
OPENAI_API_KEY=your_api_key_here npm start
```

По умолчанию сервер откроется здесь:

```text
http://localhost:8000
```

Можно выбрать модель:

```bash
OPENAI_API_KEY=your_api_key_here OPENAI_MODEL=gpt-4.1-mini npm start
```
